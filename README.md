# Data Analyst Jobs in Germany

This project analyzes Data Analyst job postings in Germany using data collected from the Adzuna API.

The goal is to understand:
- how many Data Analyst vacancies are available,
- where they are located,
- how often salary information is provided,
- which cities and companies appear most often,
- how many positions seem suitable for junior or entry-level candidates.

## Tools

- Python
- pandas
- Adzuna API
- Power BI
- Git / GitHub

## Project structure

- `src/` — Python scripts for data collection and preparation
- `notebooks/` — exploratory analysis
- `reports/screenshots/` — dashboard screenshots
- `reports/powerbi/` — Power BI notes or exported PDF/screenshots
- `data/` — local data files, not uploaded to GitHub

## Dashboard

The Power BI dashboard includes KPI cards, city distribution, salary availability, and a map of job postings in Germany.
